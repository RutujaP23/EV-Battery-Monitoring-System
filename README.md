This project focuses on the design and implementation of a low-cost EV Battery Monitoring System using a microcontroller-based approach.

## Project Title
**Designing of EV Battery Monitoring System**

## Objective
To monitor critical EV battery parameters such as:
- Voltage
- Current
- Temperature
- State of Charge (SOC)
- State of Health (SOH)

## Hardware Components
- Arduino / ESP32
- Voltage Sensor Module
- ACS712 Current Sensor
- DS18B20 Temperature Sensor
- 16x2 I2C LCD
- EV Battery Pack

## Software & Tools
- Embedded Python (MicroPython)
- Arduino IDE / Thonny
- GitHub for version control

## Features
- Real-time battery parameter monitoring
- SOC and SOH estimation
- LCD display output
- Low-cost and scalable solution

## Applications
- Electric Vehicles
- Battery testing labs
- Educational and research use
